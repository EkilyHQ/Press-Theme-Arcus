# About Mira Vale Studio

Mira Vale Studio Notes is a fictional practice journal from a two-person residential architecture studio in Portland, Maine. The demo reads like a real studio site: project diaries, material tests, client-facing field notes, and short essays about how rooms become legible over time. Names, addresses, budgets, and commissions are invented for the theme demo. Visual source material is public-domain or CC0-compatible and is cited in each article so the site can exercise Press source notes without pretending to document real projects.

This demo uses the `arcus` theme and fictional editorial content generated for theme testing.
