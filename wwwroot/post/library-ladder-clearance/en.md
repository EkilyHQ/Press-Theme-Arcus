# Library Ladder Clearance

![Library Ladder Clearance](https://commons.wikimedia.org/wiki/Special:FilePath/Library_ladder.jpg)

The ladder wanted to be beautiful. The code review wanted it to be measurable. Both were right.

We adjusted the rail height after the cardboard mockup showed that the turning radius worked on paper but not with a person carrying books. The correction was small: two inches out, one shelf bay over.

This is the sort of post a real practice keeps because future projects benefit from yesterday's awkward mockup.
